# Changelog

All notable changes to this package will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-10

### Added
- Initial release
- Scene view editing with position handles
- Support for Vector3 and float3 types
- Array and List support with visual connections
- Custom color support per waypoint field
- Grid snapping with Ctrl key (0.5 unit increments)
- Inspector property drawer integration
- Efficient caching system for reflection
- Optimized math utilities with aggressive inlining
- Sample scene demonstrating basic usage

### Features
- `[Waypoint]` attribute for marking editable fields
- Custom RGB color specification
- Automatic dotted line connections between array elements
- Local/global space transformation support
- Scene view labels for waypoint identification
- Compatible with Unity 2021.3+
